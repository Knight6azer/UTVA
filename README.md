# Vegetable Vendor Management System (UTVA)

A streamlined CLI-based management tool designed to optimize the daily operations of a small-scale vegetable vendor. This application simplifies inventory tracking, sales logging, and financial record-keeping.

## Features
- **Inventory Management:** Add, update, and track current stock levels.
- **Sales Tracking:** Log daily transactions efficiently.
- **Transaction Processing:** Automated calculations for total daily earnings.
- **CLI Interface:** Lightweight and fast command-line interaction.
